# VBound

**VBound** is a C++ project that aims to improve upon *Minecraft* in areas where Microsoft failed the community.

## Goals

- **Better performance**
- **More content to explore**
- **Fair and fun progression**
- Community-focused design and long-term extensibility

## Vision

VBound is built with efficiency and player experience in mind, focusing on technical excellence and meaningful gameplay rather than monetization-driven decisions.
